# Airline Management System
A Java-based desktop application built using **Swing** and **AWT**, designed to manage airline operations such as login authentication, flight booking, passenger details, and more.

## 📁 Project Structure
This project contains multiple Java classes responsible for different functionalities of the Airline Management System.

- **Login.java** – Handles user authentication UI and logic.
- **JFrame-based UI components** – Forms for booking, passenger details, flight information, etc.
- **Utility classes** – Database connection, CRUD operations, exception handling, etc.

## 🔧 Technologies Used
- **Java** (JDK 8+)
- **Swing / AWT** for UI
- **JDBC** (if connected to database)
- **NetBeans / IntelliJ / Eclipse** (recommended IDE)

## 🚀 How to Run
1. Extract the project files.
2. Open the project in your preferred Java IDE (NetBeans recommended).
3. Ensure JDK is properly configured.
4. Compile and run the `Login.java` file.
5. Navigate through the UI modules.

## ✨ Features
- Login authentication
- Manage passengers
- Add/view flights
- Ticket booking
- Cancellation and updates
- User-friendly Swing UI

## 📦 Requirements
- JDK 8 or above
- NetBeans/IntelliJ/Eclipse
- MySQL (optional, if database is connected)

## 🛠 Troubleshooting
- **JCalendar error**: Ensure the `jcalendar-x.x.jar` library is added to the project libraries.
- **ClassNotFoundException**: Rebuild project and check library paths.
